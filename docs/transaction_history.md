### **История операций в кошельке**
*Модель - Транзакция.* 
*При пополнении кошелька создается экземпляр модели с типом 'replenishment'.*
*При покупке Товара создается экземпляр модели с типом 'write-off'.*
*Связь с моделью Wallet - один ко многим, через поле wallet_id.*
*С историей операций пользователь может ознакомиться по пути /wallet/history.*

#### GET /wallet/history

*Возвращает cтраницу со списком операций пополнения/списания, отсортированных по дате создания.*

*Входные данные*: Request
*Формат ответа*: (200 OK) - text/html
