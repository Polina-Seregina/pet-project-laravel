### **Описание роутов управления профилем**

#### 1. Показ профиля 
*Роут*: /profile
*Метод*: GET 
*Описание*: Возвращает профиль авторизованного пользователя.
*Входные данные*: Request
*Формат ответа*: (200 OK) - text/html


#### 2. Обновление профиля

*Роут*: /profile
*Метод*: PATCH 
*Описание*: Обновляет данные в профиле пользователя.
*Входные данные*: Request
*Формат ответа*: (302 OK)


#### 3. Удаление профиля

*Роут*: /profile
*Метод*: DELETE 
*Описание*: Удаляет профиль пользователя.
*Входные данные*: Request
*Формат ответа*: (302 OK) 

#### 4. Редактирование профиля 

*Роут*: /profile/edit
*Метод*: GET
*Описание*: Возвращает форму для редактирования данных профиля.
*Входные данные*: Request
*Формат ответа*: (200 OK) - text/html
